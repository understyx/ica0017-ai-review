AWS EC2

Amazon Elastic Compute Cloud (Amazon EC2) is a web service
that provides secure, resizable compute capacity in the cloud.
It is designed to make web-scale cloud computing easier for
developers. Amazon EC2's simple web service interface allows
you to obtain and configure capacity with minimal friction.

NITRO SYSTEM (HYPERVISOR, SEC.CHIP, CARDS)

• The Nitro hypervisor is built on a minimized and modified Linux kernel
• Launched in 2017, purpose-built hardware and software
• KVM subsystem included (responsible for programming hardware
virtualization features of the processor)
• Not a general purpose operating system kernel
• The Linux kernel portion of the Nitro hypervisor has no support for
networking of any kind
• Nitro hypervisor’s sole purposes: partitioning CPU and memory, assigning
Nitro card virtual functions to instances, monitoring, metrics, and hardware
error handling.
